# Website Artifacts

Interactive web projects and tools built for **Ode by Muno**.

## Projects

- **[8c2-sensory-wheel](./8c2-sensory-wheel/)** — Spinning wheel for exploring all 28 unique pairings of Ode's eight sensory systems. React + Vite + Canvas.
- **[ces-2026-interactive](./ces-2026-interactive/)** — Interactive sensory graph for CES 2026. Vanilla JS.

## Stack

Most projects use **Vite** for builds and deploy to **Netlify**. Print artifacts use **WeasyPrint** for PDF generation.

## Ode

Ode is an independent research platform exploring intelligence as perception across eight sensory systems.

[odebymuno.com](https://www.odebymuno.com/)
